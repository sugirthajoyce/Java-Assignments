# Java-Assignments
Java Assignments

This repository contains solutions for small assignments using Java and Spring Boot.

Assignments

1. E-Commerce Billing Engine

* Implemented using OOP concepts.
* Created Item and Bill classes.
* Calculates total billing amount based on quantity and unit price.

2. Login + Product List Application

* Email validation using Regular Expressions.
* Password validation.
* Console-based product listing system.

3. Employee Filter Program

* Stores employee data using ArrayList.
* Filters employees by salary.
* Groups employees by department.
* Sorts employees alphabetically.

4. Product CRUD API

* Developed using Spring Boot.
* REST APIs:
    * POST /product/add
    * GET /product/list
    * PUT /product/update/{id}
    * DELETE /product/delete/{id}
* Uses ArrayList as in-memory storage.

Technologies Used

* Java
* OOP Concepts
* Collections Framework
* Regular Expressions
* Spring Boot
* REST API
